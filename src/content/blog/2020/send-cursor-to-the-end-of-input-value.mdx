# Send cursor to the end of input value in React

PropTypes are a mechanism to ensure that components use the correct data type and pass the right data, and that components use the right type of props, and that receiving components receive the right type of props.

Although the process only takes a minute, it can be a pain point of React development, especially when a component takes a large number of props. That is why some developers bypass PropTypes because of the small amount of extra up-front coding required.

## PropTypes Generator Extension

Now, you can Automate Adding PropTypes to Components with VS Code's React PropTypes Generator Extension.

### Installing

[Download the React PropTypes Generator](https://marketplace.visualstudio.com/items?itemName=suming.react-proptypes-generate)

### How it work

![edwe](https://raw.githubusercontent.com/dpzxsm/react-proptypes-generate/master/ScreenShot.gif)

1. Select your Component's name
2. Press command + . (Windows is Ctrl + .) show Code Actions and select PropTypesGenerate, or press shift + command + alt + P (Windows is shift + ctrl + alt + P) in the macOS
3. Input propType to replace default type
